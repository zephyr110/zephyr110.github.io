ж·ұе…Ҙжө…еҮә Webpack

1. еҺҹзҗҶ

webpack жҳҜдёҖдёӘжЁЎеқ—жү“еҢ…жңәеҲ¶пјҢе°Ҷж №жҚ®ж–Үд»¶й—ҙзҡ„дҫқиө–е…ізі»еҜ№е…¶иҝӣиЎҢйқҷжҖҒеҲҶжһҗпјҢз„¶еҗҺе°ҶиҝҷдәӣжЁЎеқ—жҢүжҢҮе®ҡи§„еҲҷз”ҹжҲҗйқҷжҖҒиө„жәҗгҖӮ
еҪ“ webpack еӨ„зҗҶзЁӢеәҸж—¶пјҢе®ғдјҡйҖ’еҪ’ең°жһ„е»әдёҖдёӘдҫқиө–е…ізі»еӣҫпјҢе…¶дёӯеҢ…еҗ«еә”з”ЁзЁӢеәҸйңҖиҰҒзҡ„жҜҸдёӘжЁЎеқ—пјҢз„¶еҗҺе°ҶжүҖжңүиҝҷдәӣжЁЎеқ—жү“еҢ…жҲҗдёҖдёӘжҲ–иҖ…еӨҡдёӘ bundleгҖӮ

дё»иҰҒеҠҹиғҪ:

  • жү“еҢ…пјҡе°ҶеӨҡдёӘж–Үд»¶жү“еҢ…жҲҗдёҖдёӘж–Үд»¶пјҢеҮҸе°‘жңҚеҠЎеҷЁеҺӢеҠӣе’ҢдёӢиҪҪеёҰе®Ҫ
  • иҪ¬жҚўпјҡе°Ҷйў„зј–иҜ‘иҜӯиЁҖиҪ¬жҚўжҲҗжөҸи§ҲеҷЁиҜҶеҲ«зҡ„иҜӯиЁҖ
  • жҖ§иғҪдјҳеҢ–

ж ёеҝғжҰӮеҝө:

  • Entry

    вҖң

    вјҠвјқиө·зӮ№(entry point)жҢҮзӨә webpack еә”иҜҘдҪҝвҪӨе“ӘдёӘжЁЎеқ—,жқҘдҪңдёәжһ„е»әе…¶еҶ…йғЁдҫқиө–еӣҫзҡ„ејҖе§ӢгҖӮ
    иҝӣвјҠвјҠвјқиө·зӮ№еҗҺ,webpack дјҡжүҫеҮәжңүе“ӘдәӣжЁЎеқ—е’Ңеә“жҳҜвјҠвјқиө·зӮ№пјҲзӣҙжҺҘе’Ңй—ҙжҺҘпјүдҫқиө–зҡ„гҖӮ
    жҜҸдёӘдҫқиө–йЎ№йҡҸеҚіиў«еӨ„зҗҶ,жңҖеҗҺиҫ“еҮәеҲ°з§°д№Ӣдёә bundles зҡ„вҪӮд»¶дёӯгҖӮ

  • Output

    вҖң

    output еұһжҖ§е‘ҠиҜү webpack еңЁе“ӘвҫҘиҫ“еҮәе®ғжүҖеҲӣе»әзҡ„ bundles,д»ҘеҸҠеҰӮдҪ•е‘ҪеҗҚиҝҷдәӣвҪӮд»¶,й»ҳи®ӨеҖјдёә ./distгҖӮ
    еҹәжң¬дёҠ,ж•ҙдёӘеә”вҪӨзЁӢеәҸз»“жһ„,йғҪдјҡиў«зј–иҜ‘еҲ°дҪ жҢҮе®ҡзҡ„иҫ“еҮәи·Ҝеҫ„зҡ„вҪӮд»¶еӨ№дёӯгҖӮ

  • Module

    вҖң

    жЁЎеқ—,еңЁ Webpack вҫҘвјҖеҲҮзҡҶжЁЎеқ—,вјҖдёӘжЁЎеқ—еҜ№еә”зқҖвјҖдёӘвҪӮд»¶гҖӮWebpack дјҡд»Һй…ҚзҪ®зҡ„ Entry ејҖе§ӢйҖ’еҪ’жүҫеҮәжүҖжңүдҫқиө–зҡ„жЁЎеқ—гҖӮ

  • Chunk

    вҖң

    д»Јз Ғеқ—,вјҖдёӘ Chunk з”ұеӨҡдёӘжЁЎеқ—з»„еҗҲвҪҪжҲҗ,вҪӨдәҺд»Јз ҒеҗҲе№¶дёҺеҲҶеүІгҖӮ

  • Loader

    вҖң

    loader и®© webpack иғҪеӨҹеҺ»еӨ„зҗҶйӮЈдәӣвҫ® JavaScript вҪӮд»¶пјҲwebpack вҫғиә«еҸӘзҗҶи§Ј JavaScriptпјүгҖӮ
    loader еҸҜд»Ҙе°ҶжүҖжңүзұ»еһӢзҡ„вҪӮд»¶иҪ¬жҚўдёә webpack иғҪеӨҹеӨ„зҗҶзҡ„жңүж•ҲжЁЎеқ—,з„¶еҗҺдҪ е°ұеҸҜд»ҘеҲ©вҪӨ webpack зҡ„жү“еҢ…иғҪвј’,еҜ№е®ғд»¬иҝӣвҫҸеӨ„зҗҶгҖӮ
    жң¬иҙЁдёҠ,webpack loader е°ҶжүҖжңүзұ»еһӢзҡ„вҪӮд»¶,иҪ¬жҚўдёәеә”вҪӨзЁӢеәҸзҡ„дҫқиө–еӣҫпјҲе’ҢжңҖз»Ҳзҡ„ bundleпјүеҸҜд»ҘзӣҙжҺҘеј•вҪӨзҡ„жЁЎеқ—гҖӮ

  • Plugin

    вҖң

    loader иў«вҪӨдәҺиҪ¬жҚўжҹҗдәӣзұ»еһӢзҡ„жЁЎеқ—,вҪҪжҸ’д»¶еҲҷеҸҜд»ҘвҪӨдәҺжү§вҫҸиҢғеӣҙжӣҙвјҙзҡ„д»»еҠЎгҖӮ
    жҸ’д»¶зҡ„иҢғеӣҙеҢ…жӢ¬,д»Һжү“еҢ…дјҳеҢ–е’ҢеҺӢзј©,вјҖзӣҙеҲ°йҮҚж–°е®ҡд№үзҺҜеўғдёӯзҡ„еҸҳйҮҸгҖӮжҸ’д»¶жҺҘвјқеҠҹиғҪжһҒе…¶ејәвјӨ,еҸҜд»ҘвҪӨжқҘеӨ„зҗҶеҗ„з§Қеҗ„ж ·зҡ„д»»еҠЎгҖӮ

жһ„е»әжөҒзЁӢ:
Webpack зҡ„иҝҗвҫҸжөҒзЁӢжҳҜвјҖдёӘдёІвҫҸзҡ„иҝҮзЁӢ,д»ҺеҗҜеҠЁеҲ°з»“жқҹдјҡдҫқж¬Ўжү§вҫҸд»ҘдёӢжөҒзЁӢ:

  1. еҲқе§ӢеҢ–еҸӮж•°пјҡд»Һй…ҚзҪ®вҪӮд»¶е’Ң Shell иҜӯеҸҘдёӯиҜ»еҸ–дёҺеҗҲе№¶еҸӮж•°,еҫ—еҮәжңҖз»Ҳзҡ„еҸӮж•°гҖӮ
  2. ејҖе§Ӣзј–иҜ‘пјҡвҪӨдёҠвјҖжӯҘеҫ—еҲ°зҡ„еҸӮж•°еҲқе§ӢеҢ– Compiler еҜ№иұЎ,еҠ иҪҪжүҖжңүй…ҚзҪ®зҡ„жҸ’д»¶,жү§вҫҸеҜ№иұЎзҡ„ run вҪ…жі•ејҖе§Ӣжү§вҫҸзј–иҜ‘гҖӮ
  3. зЎ®е®ҡвјҠвјқпјҡж №жҚ®й…ҚзҪ®дёӯзҡ„ entry жүҫеҮәжүҖжңүзҡ„вјҠвјқвҪӮд»¶гҖӮ
  4. зј–иҜ‘жЁЎеқ—пјҡд»ҺвјҠвјқвҪӮд»¶еҮәеҸ‘,и°ғвҪӨжүҖжңүй…ҚзҪ®зҡ„ Loader еҜ№жЁЎеқ—иҝӣвҫҸзҝ»иҜ‘,еҶҚжүҫеҮәиҜҘжЁЎеқ—дҫқиө–зҡ„жЁЎеқ—,еҶҚйҖ’еҪ’жң¬жӯҘйӘӨзӣҙеҲ°жүҖжңүвјҠвјқдҫқиө–зҡ„вҪӮд»¶йғҪз»ҸиҝҮдәҶжң¬жӯҘйӘӨзҡ„еӨ„зҗҶгҖӮ
  5. е®ҢжҲҗжЁЎеқ—зј–иҜ‘пјҡеңЁз»ҸиҝҮз¬¬ 4 жӯҘдҪҝвҪӨ Loader зҝ»иҜ‘е®ҢжүҖжңүжЁЎеқ—еҗҺ,еҫ—еҲ°дәҶжҜҸдёӘжЁЎеқ—иў«зҝ»иҜ‘еҗҺзҡ„жңҖз»ҲеҶ…е®№д»ҘеҸҠе®ғд»¬д№Ӣй—ҙзҡ„дҫқиө–е…ізі»гҖӮ
  6. иҫ“еҮәиө„жәҗпјҡж №жҚ®вјҠвјқе’ҢжЁЎеқ—д№Ӣй—ҙзҡ„дҫқиө–е…ізі»,з»„иЈ…жҲҗвјҖдёӘдёӘеҢ…еҗ«еӨҡдёӘжЁЎеқ—зҡ„ Chunk,еҶҚжҠҠжҜҸдёӘ Chunk иҪ¬жҚўжҲҗвјҖдёӘеҚ•зӢ¬зҡ„вҪӮд»¶еҠ вјҠеҲ°иҫ“еҮәеҲ—иЎЁ,иҝҷжӯҘжҳҜеҸҜд»Ҙдҝ®ж”№иҫ“еҮәеҶ…е®№зҡ„жңҖеҗҺжңәдјҡгҖӮ
  7. иҫ“еҮәе®ҢжҲҗпјҡеңЁзЎ®е®ҡеҘҪиҫ“еҮәеҶ…е®№еҗҺ,ж №жҚ®й…ҚзҪ®зЎ®е®ҡиҫ“еҮәзҡ„и·Ҝеҫ„е’ҢвҪӮд»¶еҗҚ,жҠҠвҪӮд»¶еҶ…е®№еҶҷвјҠеҲ°вҪӮд»¶зі»з»ҹгҖӮ
    еңЁд»ҘдёҠиҝҮзЁӢдёӯ,Webpack дјҡеңЁзү№е®ҡзҡ„ж—¶й—ҙзӮ№вјҙж’ӯеҮәзү№е®ҡзҡ„дәӢд»¶,жҸ’д»¶еңЁзӣ‘еҗ¬еҲ°ж„ҹе…ҙи¶Јзҡ„дәӢд»¶еҗҺдјҡжү§вҫҸзү№е®ҡзҡ„йҖ»иҫ‘,е№¶дё”жҸ’д»¶еҸҜд»Ҙи°ғвҪӨ Webpack жҸҗдҫӣзҡ„ API ж”№еҸҳ Webpack зҡ„иҝҗвҫҸз»“жһңгҖӮ
